# Ad Placement Guide

This guide covers all available ad positions and recommended image dimensions for creating ads in the admin panel.

## Available Ad Positions

| Position       | Label          | Description             |
| -------------- | -------------- | ----------------------- |
| SIDEBAR        | Sidebar        | Standard sidebar ad     |
| BANNER         | Banner         | Banner ad               |
| IN_ARTICLE     | In Article     | Within article content  |
| SIDEBAR_TOP    | Sidebar Top    | Top of right sidebar    |
| SIDEBAR_BOTTOM | Sidebar Bottom | Bottom of right sidebar |
| HOME_MIDDLE    | Home Middle    | Middle of homepage      |
| ARTICLE_DETAIL | Article Detail | Article detail pages    |

## Recommended Image Dimensions

| Position                                   | Dimensions                                  | Aspect Ratio |
| ------------------------------------------ | ------------------------------------------- | ------------ |
| **BANNER / HOME_TOP**                      | 728 × 90px                                  | 8:1          |
| **HOME_MIDDLE**                            | 728 × 90px (desktop) / 300 × 250px (mobile) | 8:1 or 1.2:1 |
| **SIDEBAR / SIDEBAR_TOP / SIDEBAR_BOTTOM** | 300 × 250px                                 | 1.2:1        |
| **IN_ARTICLE**                             | 728 × 90px or 300 × 250px                   | 8:1 or 1.2:1 |
| **ARTICLE_DETAIL**                         | 728 × 90px or 300 × 250px                   | 8:1 or 1.2:1 |

## Supported File Types

- Images: JPG, PNG, WEBP
- Animated: GIF
- Scripts: JavaScript (for ad networks like Google Adsense)

## Best Practices

1. **Use the correct aspect ratio** - Sidebar ads should be 300px wide, banner ads 728px wide
2. **Keep file sizes small** - Aim for under 150KB for faster page loads
3. **Create both sizes** - For responsive positions like IN_ARTICLE, consider creating multiple versions
4. **Test on mobile** - Many users browse on mobile devices
5. **Add click-through URL** - Make sure to add a valid link URL for tracking ad clicks

## Ad Click Tracking

When users click on ads, the system automatically tracks the click. You can view ad performance in the analytics.
